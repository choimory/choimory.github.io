# volatile 키워드의 역할

- `volatile` 키워드는 멀티쓰레드 환경에서 변수의 값을 여러 쓰레드가 **일관되게 읽고 쓸 수 있도록 보장**함.
    - 변수의 값을 **메인 메모리**에서 직접 읽고 쓰도록 하여, 한 쓰레드에서 변경된 값이 다른 쓰레드에서도 즉시 반영됨을 보장함.
- `volatile`은 **명령어 재정렬**을 방지함.
    - 컴파일러나 CPU가 성능 최적화를 위해 명령어의 실행 순서를 바꾸는 것을 막아줌으로써, 변수의 상태가 예측 가능한 방식으로 일관되게 유지되도록 함.
- `volatile` 키워드는 변수의 **메모리 가시성**과 **명령어 순서 보장**을 제공하여, 여러 쓰레드가 동시에 접근하는 변수에 대해 안전한 사용을 가능하게 함.
